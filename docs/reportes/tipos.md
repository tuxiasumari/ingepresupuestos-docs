# Tipos de reporte

IngePresupuestos genera **13 tipos de reporte**, todos consistentes entre PDF, Excel, Word, ODS y ODT.

| Reporte | Qué contiene |
|---------|--------------|
| **Memoria Descriptiva** | El documento descriptivo del proyecto. |
| **Resumen Ejecutivo** | Presupuesto total, estructura y principales cifras. |
| **Presupuesto** | El presupuesto detallado con partidas, metrados y precios. |
| **Desagregado del Pie** | El detalle de gastos generales y demás rubros. |
| **Análisis de Costos (ACU)** | El análisis de costos unitarios de cada partida. |
| **Hoja de Metrados** | La planilla de sustento de metrados. |
| **Insumos** | La relación de insumos consolidada. |
| **Especificaciones Técnicas** | Las especificaciones de cada partida. |
| **Cronograma — Gantt** | El diagrama de Gantt. |
| **Cronograma Valorizado** | La valorización por período. |
| **Cronograma de Adquisiciones** | Las compras por período. |
| **Curva S** | El avance acumulado. |
| **Reporte Completo** | Todo el expediente en un solo documento. |

A esto se suma la **Fórmula Polinómica**, que también se exporta.

!!! tip "Tamaño de papel"
    Los reportes normales salen en **A4**. Los cronogramas anchos (valorizado, adquisiciones) y la Curva S permiten elegir el papel. El **Gantt** ajusta el tamaño automáticamente (de A4 a A0) según la cantidad de tareas.
