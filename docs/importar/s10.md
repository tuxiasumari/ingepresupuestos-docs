# Importar desde S10

Los presupuestos de **S10** se guardan en archivos `.S2K`, `.bkf` o `.bak` (backups de base de datos). IngePresupuestos los importa con ayuda de **IngeConverter**, una herramienta gratuita que se descarga junto al programa.

## Qué necesitas

- El archivo de backup de S10: **`.S2K`**, **`.bkf`** o **`.bak`**.
- **IngeConverter** (incluido gratis). Si no lo tienes, el programa te ofrece descargarlo cuando eliges importar de S10.

## Pasos

1. En IngePresupuestos: **Importar → S10**.
2. Selecciona tu archivo `.S2K` / `.bkf` / `.bak`.
3. IngeConverter procesa el backup y lo convierte a un proyecto de IngePresupuestos.
4. Confirma. El proyecto aparece en tu lista con sus partidas, ACU e insumos.

!!! info "Primera vez en Windows"
    IngeConverter usa una base de datos local (SQL Server LocalDB) para leer los backups de S10. La primera vez puede pedirte instalar un componente; sigue las indicaciones y, si te lo solicita, **reinicia** antes de volver a importar.

## Versiones de S10 soportadas

Se soportan los backups de las versiones modernas de S10. Los backups muy antiguos (formato de S10 2000 y anteriores) **no** están soportados por ahora.

!!! tip "¿Tu backup no abre?"
    Escríbenos por [WhatsApp](https://wa.me/51998839090) con el detalle del archivo. Estamos ampliando la compatibilidad y tu caso nos ayuda a mejorar el conversor.
