# Elaborar el presupuesto

Esta sección cubre todo el trabajo dentro de un proyecto: armar la estructura de partidas, analizar sus costos, consolidar insumos, sustentar metrados, redactar especificaciones y memoria, ubicar la obra y configurar el pie de presupuesto.

<div class="grid cards" markdown>

-   :material-state-machine: **[Estados del proyecto](estados.md)**

    En elaboración, revisión, aprobado o ejecutado: qué se puede editar en cada uno.

-   :material-format-list-numbered: **[Partidas y subpresupuestos](partidas.md)**

    La estructura del presupuesto: componentes, títulos y partidas.

-   :material-calculator-variant: **[Análisis de Costos Unitarios](acu.md)**

    Mano de obra, materiales y equipo de cada partida.

-   :material-package-variant: **[Insumos](insumos.md)**

    Todos los recursos del proyecto, consolidados y con precio único.

-   :material-ruler-square: **[Metrados](metrados.md)**

    La planilla de sustento, incluido el acero por diámetro.

-   :material-text-box: **[Especificaciones técnicas](especificaciones.md)**

    La descripción técnica de cada partida.

-   :material-file-document-edit: **[Memoria descriptiva](memoria.md)**

    El documento del proyecto, redactado con ayuda de IA.

-   :material-map-marker: **[Ubicación y mapa](mapa.md)**

    Geolocaliza la obra y captura coordenadas UTM.

-   :material-table-large: **[Pie de presupuesto](pie.md)**

    Gastos generales, utilidad, IGV y demás rubros.

</div>

!!! note "El estado del proyecto controla qué se puede editar"
    Un proyecto puede estar **En elaboración**, **En revisión**, **Aprobado** o **Ejecutado**. Según el estado, algunas partes quedan protegidas contra edición. Revisa **[Estados del proyecto](estados.md)** para ver qué se bloquea en cada uno y cómo cambiarlo.
